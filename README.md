# Todo App

Educational JavaScript project: a simple Todo application built with vanilla JavaScript.
Focus: DOM manipulation, event handling, and client-side data persistence.

**Live Demo:** <https://mariaroim.com/todo-app/>  
**Repository:** https://github.com/mariaroim-dev/todo-app

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks are saved in LocalStorage and restored after page reload
- UI updates without page reload (DOM manipulation)

## Getting Started
1. Clone the repository
2. Open `index.html` in your browser

## Project Structure
- `index.html` — main page
- `todo-app.js` — application logic
